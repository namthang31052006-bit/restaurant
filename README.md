# FreshBite Restaurant Website

This is a responsive restaurant website created for a Coursera portfolio project.

## Files
- `index.html` - main webpage
- `style.css` - responsive styling
- `script.js` - interactive features

## Features
1. Responsive design for mobile and larger screens.
2. Menu search and category filter.
3. Dark mode toggle.
4. Reservation form with confirmation message.
5. Back to top button.
6. Skip to main content accessibility link.

## How to host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select the `main` branch and `/root`.
6. Save and wait for the GitHub Pages link.
